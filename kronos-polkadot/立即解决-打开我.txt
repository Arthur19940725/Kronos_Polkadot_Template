  ✓ 提供针对性的解决步骤

方法2：手动创建账户
----------------------------------
继续阅读下方说明 ↓

================================================================

【Polkadot.js Extension - 5个步骤】

第1步：找到扩展
  1. 看浏览器右上角
  2. 找红色圆点图标（Polkadot{.js}）
  3. 点击打开

第2步：点击加号
  1. 在扩展窗口右上角
  2. 点击 [+] 按钮

第3步：创建新账户
  1. 选择 "Create new account"
  2. 或 "Create account from new seed"

第4步：抄写助记词
  ⚠️  会显示 12 个英文单词
  ⚠️  用纸笔抄写下来！
  ⚠️  这是恢复账户的唯一方式！
  ⚠️  不要截图！

第5步：设置信息
  1. 账户名称: 随便取（如 Kronos）
  2. 密码: 设置强密码
  3. 确认密码: 再输一次
  4. 点击 "Add the account"
  5. 完成！

================================================================

【OKX Wallet - 6个步骤】

第1步：打开 OKX
  1. 点击浏览器右上角 OKX 图标（橙色）
  2. 打开钱包

第2步：创建钱包
  1. 首次打开会提示
  2. 点击 "创建钱包" 或 "Create wallet"

第3步：设置密码
  1. 输入密码
  2. 确认密码
  3. 勾选同意条款
  4. 点击创建

第4步：备份助记词
  1. 点击 "显示" 查看助记词
  2. 用纸笔抄写 12 个单词
  3. 按提示验证
  4. 确认备份完成

第5步：切换到 Polkadot 网络（重要！）
  1. 点击顶部网络名称（如 "Ethereum Mainnet"）
  2. 在搜索框输入: Polkadot
  3. 选择 "Polkadot" 或 "Westend"
  4. 等待切换完成
  5. 确认顶部显示 "Polkadot"

第6步：完成！
  1. 现在能看到 Polkadot 账户
  2. 地址以 5 开头
  3. 可以使用了！

================================================================

【创建后如何验证】

Polkadot.js：
  1. 打开扩展
  2. 应该能看到：
     ┌─────────────────┐
     │ ● Kronos Account│  ← 您的账户名
     │   5Fg9...8K3d   │  ← 地址
     └─────────────────┘

OKX Wallet：
  1. 打开扩展
  2. 应该能看到：
     ┌─────────────────┐
     │ Polkadot [▼]    │  ← 确认是 Polkadot
     │ Account 1       │  ← 账户名
     │ 0.0000 DOT      │  ← 余额
     │ 5Fg9...8K3d     │  ← 地址（5开头）
     └─────────────────┘

【创建账户后，连接到 Kronos】

步骤1：

步骤2：

  → 成功！开始使用！

【常见问题】

Q: 我创建了账户，为什么还是提示没有？
A: 
  1. 刷新 DApp 页面（F5）
  2. 检查钱包扩展中能否看到账户
  3. (OKX用户)确认已切换到 Polkadot 网络
  4. 重新点击 "Connect Wallet"

Q: OKX 钱包显示的是以太坊地址？
A: 
  您需要切换网络！
  1. 点击 OKX 顶部的网络名称
  2. 搜索 Polkadot
  3. 选择 Polkadot 网络
  4. 地址应该变成 5 开头

Q: 助记词丢了怎么办？
A: 
  无法恢复！只能重新创建新账户。
  这就是为什么必须备份的原因！

【安全提醒】

助记词 = 您的钱包
丢失 = 永久失去
泄露 = 资产被盗

必须：
  ✓ 纸笔抄写
  ✓ 保存安全的地方
  ✓ 多抄几份备份

不要：
  × 截图
  × 保存在电脑
  × 告诉任何人
  × 发送给客服
