QUICK START (5 minutes)
================================================================

Step 1: Install a Wallet Extension (Choose one)
------------------------------------------------
Option A - Polkadot.js (Recommended for beginners)
  1. Visit: https://polkadot.js.org/extension/
  2. Click "Add to Chrome"
  3. Done!

Option B - OKX Wallet (Recommended for multi-chain)
  1. Visit: https://www.okx.com/web3
  2. Download and install
  3. Create wallet
  4. Switch to Polkadot network

Step 2: Create an Account
------------------------------------------------
1. Click the wallet icon (top-right of browser)
2. Click "+" or "Create Account"
3. IMPORTANT: Write down 12 words on paper!
4. Set account name and password
5. Done!

Step 3: Connect to Kronos
------------------------------------------------
1. Visit: http://localhost:3000
2. Click "Connect Wallet" (top-right)
3. Select your wallet
4. Choose account and authorize
5. Start predicting!

ABOUT METAMASK
================================================================

MetaMask is an Ethereum wallet, doesn't directly support Polkadot.

EASIEST SOLUTION:
  → Use OKX Wallet instead
    (supports both Ethereum and Polkadot)

IF YOU MUST USE METAMASK:
  → Install Polkadot Snap: https://polkagate.xyz/snap
    (requires extra setup)

RECOMMENDED:
  → Keep MetaMask for Ethereum
  → Install OKX Wallet for Polkadot
  → Both wallets can coexist

SECURITY WARNING
================================================================

Your recovery phrase (12 words) = Your entire wallet

DO:
  ✓ Write it on paper
  ✓ Store in safe place
  ✓ Use strong password
  ✓ Test with small amounts

DON'T:
  × Share with anyone
  × Screenshot it
  × Save on computer
  × Use weak password

Lost recovery phrase = Lost wallet forever!
